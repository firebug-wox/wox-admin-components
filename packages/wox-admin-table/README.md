# wox-admin-table

## API

* [官方文档API](http://ant.design/components/table-cn/#API)

* 新增API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| rowColorDifferent | 是否设置单双行背景颜色区分 | boolean | false |

> 注意：如果设置了 `rowClassName` ， `rowColorDifferent`将会失效

## Usage

```
import WoxTable from 'wox-admin-table';

render(
  <WoxTable />,
  rootEle
);
```
